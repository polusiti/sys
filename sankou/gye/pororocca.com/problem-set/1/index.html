第 1 問 チュートリアル(閲覧・解答編) writer : pororocca

問題を閲覧・解答する

左上の問題一覧から作成された問題を閲覧することができます。解答の提出はログインした状態でのみ行えます。

問題詳細ページの右上に4つ(3つ)のボタンがあります。それぞれの機能は以下の通りです。

問題詳細ページでは自分の解答や他の人の解答を参照したり、問題についての質問をしたり、解説を見たりすることができます。
ジャッジ形式が自動ジャッジの問題は正解するまでは解説や他の人の解答をみることはできません。

この問題は解答の練習に使うことができます。自動ジャッジ形式で正規表現を用いて「(ネコ)|(ねこ)」という解答が設定されていますので、この場合は「ネコ」もしくは「ねこ」と解答していただくと正解となります。

第 2 問 チュートリアル(問題投稿編) writer : pororocca

問題を投稿する

問題投稿ページから投稿することができます。
なお、問題投稿はログインしていただいた状態でのみ行えます。
すぐに投稿することもできますし、下書きとして保持しておくこともできます。
ページ右上のご自身のアカウントをクリックすることで下書きにアクセスすることができます。
一度ページを離れてしまうと書いた内容は全て消えてしまうのでその際には下書きとして一旦保存していただくと良いかもしれません。
また、問題編集ページからは、一度公開した問題を更新したり、再度下書きに戻したりすることもできます。

問題中に画像を挿入したい場合には枠内にドラッグ&ドロップをしていただくとできます。

正解の設定方法

なお、自動ジャッジの際には正規表現を用いることができます。
その際、数式を解答とする場合には正規表現で用いられる記号と数式の記号が衝突してしまい、
正確なジャッジが行われないことがあります。
そのため、数式を解答とする際には以下の例のように書いてください。
例) 「x^2 + 2x + 1」-> 「x\^2 \+ 2x \+ 1」

今こちらの問題では「x\^2 \+ 2x \+ 1」が正解となっています。よって「x^2 + 2x + 1」と解答することで正解することができます。
ぜひ今解答してみてください。

第 3 問 チュートリアル(模試編) writer : pororocca

模試を作成する

模試機能とは、複数の問題をまとめて、1つのセットにすることができる機能です。

模試一覧より模試の枠組みを作成することができ、問題詳細ページの右上の「模試を作る」タブを開くと
問題を追加することのできる模試の一覧が表示されます。
追加したい模試の追加(ブックマークと同じマーク)を押すことで問題を模試に追加することができます。

模試の題名をクリックすると模試に含まれる問題をみることができます。
その際の右上より、ご自身が作成した模試を編集することができます。
